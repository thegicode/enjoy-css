#Flex


## Initial value
	- flex-grow : 0
	- flex-shrink : 1
	- flex-basis : auto


## Short hand
	> Formal synstax
		none | [ <'flex-grow'> <'flex-shrink'> || <'flex-basis'> ]

	> flex : intial
		flex : 0 1 auto

	> flex : auto
		flex : 1 1 auto

	> flex : 1
		flex : 1 1 0

	> flex : none 
		0 0 auto
